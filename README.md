# Motor-B-squeda-Reglamento
# 📌 Descripción

Este proyecto implementa un motor de búsqueda basado en TF-IDF que permite encontrar artículos relevantes de un reglamento universitario a partir de consultas en lenguaje natural.
El sistema analiza la frecuencia de palabras (TF) y su importancia en el conjunto de documentos (IDF) para calcular qué tan relevante es cada artículo frente a una consulta del usuario.

# 🎯 Objetivo

Permitir a un estudiante buscar información específica del reglamento (becas, sanciones, cancelaciones, etc.) obteniendo los documentos más relevantes.

# ▶️ Instrucciones de ejecución

1. Clonar el repositorio: git clone 
2. Entrar a la carpeta:
3. Ejecutar el programa: python main.py

# 🔎 Funcionamiento
El sistema realiza los siguientes pasos:

1. Preprocesa los textos (tokenización y limpieza)
2. Calcula TF (frecuencia de términos)
3. Calcula IDF (importancia global)
4. Calcula el score TF-IDF
5. Ordena los documentos por relevancia
